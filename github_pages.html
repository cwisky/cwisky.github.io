GitHub Pages로 정적 웹사이트 만들기

다음은 HTML, CSS, JavaScript로 만든 정적 웹사이트를 GitHub Pages를 통해 무료로 배포하는 방법입니다.

✅ 단계별 절차

  1. GitHub 저장소 만들기
    GitHub에 로그인 후 New repository를 클릭하고, 공개(Public) 저장소를 생성합니다.
  2. 웹 파일 준비
    index.html 파일을 포함한 웹사이트 파일을 준비합니다.
  3. 파일 업로드
    웹 프로젝트 파일을 GitHub 저장소에 업로드하거나 Git을 사용해 푸시합니다.
  4. GitHub Pages 설정
    저장소 페이지에서 Settings > Pages로 이동한 후,
    Branch: main, Directory: /(root)를 선택합니다.
  5. 웹사이트 접속
    잠시 기다리면 배포 주소가 생성되며, 아래와 같은 주소로 접속할 수 있습니다:
    https://사용자이름.github.io/저장소이름/

📌 주의사항

  • 반드시 index.html 파일이 있어야 첫 페이지로 인식됩니다.
  • 정적 웹사이트만 가능하며 서버 처리는 지원하지 않습니다.
  • 저장소 이름을 사용자이름.github.io로 만들면 루트 도메인에서 실행됩니다.
