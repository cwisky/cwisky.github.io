🎯 CSS 다양한 선택자 요약

📌 1. 기본 선택자

  • 전체 선택자: * → 모든 요소에 적용
  • 태그 선택자: p, div 등
  • 클래스 선택자: .클래스명
  • ID 선택자: #아이디명
* { margin: 0; padding: 0; } p { color: blue; } .container { width: 100%; } #header { background-color: lightgray; }

🔗 2. 결합자 선택자 (Combinators)

  • A B: A 요소 안의 모든 B 요소 (자손 선택자)
  • A > B: A 요소의 **직계 자식** B 선택
  • A + B: A 요소 **바로 다음 형제** B
  • A ~ B: A 이후의 **모든 형제** B
div p { color: red; } ul > li { font-weight: bold; } h1 + p { margin-top: 0; } h2 ~ p { color: gray; }

🧪 3. 속성 선택자

  • [속성]: 해당 속성이 있는 요소
  • [속성="값"]: 정확히 일치하는 경우
  • [속성^="값"]: 값으로 시작
  • [속성$="값"]: 값으로 끝남
  • [속성*="값"]: 값 포함
input[type="text"] { border: 1px solid #ccc; } a[href^="https"] { color: green; } img[alt$=".jpg"] { border-radius: 5px; }

🎯 4. 가상 클래스 선택자 (Pseudo-classes)

  • :hover → 마우스를 올렸을 때
  • :focus → 입력 필드 선택 시
  • :nth-child(n) → n번째 자식
  • :first-child, :last-child
a:hover { text-decoration: underline; } input:focus { outline: 2px solid blue; } li:nth-child(2) { color: red; }

🎨 5. 가상 요소 선택자 (Pseudo-elements)

  • ::before: 요소 앞에 콘텐츠 삽입
  • ::after: 요소 뒤에 콘텐츠 삽입
  • ::first-line: 첫 줄 스타일
p::before { content: "👉 "; color: orange; } p::after { content: " ✅"; color: green; }

📌 6. 선택자 조합 예시

ul.menu > li:first-child a:hover { color: crimson; font-weight: bold; }

※ 메뉴의 첫 번째 항목에 마우스를 올렸을 때 텍스트 강조
