💻 JavaScript 소개 및 기본 문법

📌 1. JavaScript란?

  • 웹 페이지에 **동적인 기능**을 추가하는 프로그래밍 언어
  • HTML, CSS와 함께 **웹의 3대 핵심 기술**
  • 사용자 입력 처리, 애니메이션, 데이터 처리 등 가능

🎯 2. JavaScript의 역할

  • 버튼 클릭, 폼 제출 등 **이벤트 처리**
  • HTML 요소에 접근하고 **동적으로 변경**
  • 서버와 통신 (AJAX, API 호출)
  • 유효성 검사, 계산, 애니메이션 등 다양
// 예: 버튼 클릭 시 배경색 변경 document.querySelector("button").addEventListener("click", () => { document.body.style.backgroundColor = "lightblue"; });

🔤 3. 변수 선언 (var, let, const)

  • var: 오래된 방식, 범위 제한 약함
  • let: 변경 가능한 변수 (블록 범위)
  • const: 상수, 변경 불가
var name = "Alice"; let age = 25; const pi = 3.14;

📦 4. 주요 자료형 (Data Types)

  • 문자열(String): "Hello", 'JavaScript'
  • 숫자(Number): 10, 3.14
  • 불리언(Boolean): true, false
  • undefined: 값이 정의되지 않음
  • null: 명시적인 '없음'
  • 객체(Object): 키-값 쌍의 집합
  • 배열(Array): 여러 값을 순서대로 저장
let name = "철수"; // 문자열 let score = 90.5; // 숫자 let isPassed = true; // 불리언 let user = { name: "영희", age: 22 }; // 객체 let fruits = ["사과", "바나나", "포도"]; // 배열

💡 5. typeof 연산자

  • 값의 자료형을 확인할 수 있는 연산자
console.log(typeof "hello"); // string console.log(typeof 123); // number console.log(typeof true); // boolean console.log(typeof null); // object (주의!) console.log(typeof undefined); // undefined
