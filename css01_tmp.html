CSS 소개 및 적용 방법

CSS란?

CSS(Cascading Style Sheets)는 HTML 요소의 스타일을 지정하는 스타일 시트 언어입니다. 글꼴, 색상, 여백, 정렬 등 웹 페이지의 디자인을 제어합니다.

CSS 적용 방법

  1. 인라인 방식
    <h1 style="color:blue;">제목</h1>
  2. 내부 스타일 시트 (Internal) 
    <style>
h1 {
  color: blue;
}
</style>
  3. 외부 스타일 시트 (External)

    CSS 파일을 별도로 작성하여 링크하는 방식입니다.

    <link rel="stylesheet" href="style.css">

    style.css 파일 내용 예시:

    body {
  background-color: #f0f0f0;
  color: #333;
}
