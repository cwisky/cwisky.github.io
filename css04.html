🎨 CSS 배경, 테두리, Flexbox 기본

🌈 1. 배경 (Background)

  • background-color: 배경 색상
  • background-image: 배경 이미지
  • background-repeat: 반복 여부
  • background-size: 크기 조정
  • background-position: 위치 설정
.box { background-color: #e0f7fa; background-image: url("bg.png"); background-repeat: no-repeat; background-size: cover; background-position: center; }

🧱 2. 테두리 (Border)

  • border: 테두리 전체 설정
  • border-width, border-style, border-color
  • border-radius: 테두리 둥글게
.card { border: 2px solid #4caf50; border-radius: 8px; padding: 20px; }

📦 3. Flexbox 기본 개념

  • 요소를 가로 방향, 세로 방향 정렬에 유리하게 배치
  • display: flex;로 부모 요소에 설정
  • 축 정렬 속성:
    • justify-content: 주 축 정렬 (가로)
    • align-items: 교차 축 정렬 (세로)
    • flex-direction: 방향 설정 (row, column)
.container { display: flex; flex-direction: row; justify-content: center; align-items: center; }

✅ Flexbox 정렬 예시

.flex-center { display: flex; justify-content: center; align-items: center; height: 200px; border: 2px dashed #aaa; }

※ 위 설정은 요소를 수평/수직 모두 중앙에 정렬합니다.
