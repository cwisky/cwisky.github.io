📝 Form 태그 내 입력 요소 요약

📌 1. <form> 태그란?

  • 사용자 입력 데이터를 서버로 전송하는 폼 영역
  • action, method 속성 포함
<form action="/submit" method="post"> ...입력 필드들... </form>

✍️ 2. <input> 태그 - 다양한 타입

  • type="text": 일반 텍스트 입력
  • type="password": 비밀번호 입력 (숨김)
  • type="email": 이메일 형식 검사
  • type="number": 숫자만 입력
  • type="checkbox": 체크박스
  • type="radio": 라디오 버튼 (하나만 선택)
  • type="file": 파일 업로드
<input type="text" name="username"> <input type="password" name="password"> <input type="checkbox" name="agree"> 동의 <input type="radio" name="gender" value="male"> 남성 <input type="radio" name="gender" value="female"> 여성 <input type="file" name="resume">

📝 3. <textarea> 태그

  • 여러 줄 텍스트 입력을 위한 영역
  • 행(rows), 열(cols) 설정 가능
<textarea name="comment" rows="4" cols="40"></textarea>

🔽 4. <select> & <option>

  • 드롭다운 목록 구성
  • <option>으로 항목 나열
<select name="country"> <option value="kr">대한민국</option> <option value="us">미국</option> </select>

🔘 5. <button> 태그

  • 폼 전송 또는 일반 버튼으로 사용
  • type="submit" (전송), type="reset" (초기화)
<button type="submit">제출</button> <button type="reset">초기화</button>

🏷️ 6. <label> 태그

  • 입력 필드의 설명을 연결해 접근성 향상
  • for 속성은 해당 id와 일치
<label for="email">이메일:</label> <input type="email" id="email" name="email">

📋 7. 종합 예시

<form action="/submit" method="post"> <label for="name">이름:</label> <input type="text" id="name" name="name"><br> <label for="email">이메일:</label> <input type="email" id="email" name="email"><br> <label> <input type="checkbox" name="agree"> 이용약관 동의 </label><br> <button type="submit">전송하기</button> </form>
