📱 반응형 웹 & 미디어쿼리 요약

🌐 1. 반응형 웹이란?

  • 사용자의 화면 크기에 따라 자동으로 레이아웃이 조정되는 웹 디자인
  • PC, 태블릿, 스마트폰 등 다양한 디바이스에 최적화
  • 화면 확대 없이도 편리하게 콘텐츠 이용 가능

💡 2. 반응형 웹의 핵심 요소

  • viewport 메타 태그 설정
  • 유동적인 크기 (%, vw, vh 등)
  • 미디어 쿼리를 활용한 스타일 분기
<meta name="viewport" content="width=device-width, initial-scale=1.0">

🎯 3. 미디어 쿼리(Media Query)란?

  • 화면 크기, 장치 종류에 따라 CSS 스타일을 다르게 적용
  • @media 키워드 사용
  • 주로 max-width, min-width를 사용
/* 화면 너비가 768px 이하일 때 */ @media (max-width: 768px) { .container { flex-direction: column; } }

📐 4. 미디어쿼리 작성 예시

/* 기본 스타일 (데스크탑) */ .box { width: 600px; margin: auto; } /* 태블릿용 스타일 */ @media (max-width: 992px) { .box { width: 90%; } } /* 모바일용 스타일 */ @media (max-width: 576px) { .box { width: 100%; padding: 10px; } }

🔧 5. 미디어 속성 종류

  • width: 뷰포트의 너비
  • height: 뷰포트의 높이
  • orientation: 방향 (portrait/landscape)
  • resolution: 해상도 (dpi, dppx)
  • hover: 마우스 포인터 여부

✅ 6. 정리 요약

  • 반응형 웹은 다양한 화면 크기에 자동 적응
  • 미디어 쿼리는 스타일을 화면 조건에 따라 변경
  • 모바일 퍼스트 접근 방식 권장 (min-width부터 작성)
