🧾 간단한 회원가입 폼 만들기 (검증 포함)

📌 1. 폼 구성 요소

  • 이름 (text)
  • 이메일 (email)
  • 비밀번호 (password)
  • 비밀번호 확인
  • 제출 버튼

📄 2. HTML 구조 예시

<form id="signupForm"> 이름: <input type="text" id="name" required><br> 이메일: <input type="email" id="email" required><br> 비밀번호: <input type="password" id="password" required minlength="6"><br> 비밀번호 확인: <input type="password" id="confirm" required><br> <button type="submit">회원가입</button> </form>

🛡️ 3. JavaScript로 입력 검증

  • 모든 필드가 채워졌는지 확인
  • 이메일 형식 포함 여부
  • 비밀번호와 비밀번호 확인 일치 여부
document.getElementById("signupForm").addEventListener("submit", function(e) { e.preventDefault(); // 기본 제출 방지 const name = document.getElementById("name").value.trim(); const email = document.getElementById("email").value.trim(); const password = document.getElementById("password").value; const confirm = document.getElementById("confirm").value; if (name === "" || email === "" || password === "" || confirm === "") { alert("모든 항목을 입력해주세요."); return; } if (!email.includes("@")) { alert("유효한 이메일을 입력하세요."); return; } if (password.length < 6) { alert("비밀번호는 6자 이상이어야 합니다."); return; } if (password !== confirm) { alert("비밀번호가 일치하지 않습니다."); return; } alert("회원가입이 완료되었습니다!"); });

💡 4. 추가 검증 방법 (선택)

  • 정규표현식으로 이메일 또는 비밀번호 복잡도 검사
  • 이름에 한글만 허용 등 사용자 정의 규칙 적용 가능

✅ 5. 요약

  • 폼 값은 .value로 읽고, trim()으로 공백 제거
  • 필수값, 이메일 형식, 비밀번호 일치 여부는 자바스크립트로 유효성 검사
  • e.preventDefault()로 기본 제출 막고 로직 수행
