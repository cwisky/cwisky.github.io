🧩 DOM 이해 및 조작

📌 1. DOM이란?

  • DOM (Document Object Model)은 HTML 문서를 객체로 표현한 구조
  • 문서의 각 요소를 노드(Node)로 구성
  • JavaScript를 통해 문서 내용이나 구조를 조작 가능

🧱 2. DOM 구조

  • Document 객체가 최상위 루트
  • 각 태그는 Element Node
  • 텍스트는 Text Node
document └── html ├── head └── body ├── h1 └── p

🔍 3. DOM 요소 선택

  • getElementById(): ID로 요소 찾기
  • getElementsByClassName(): 클래스명으로 찾기
  • querySelector(): CSS 선택자 방식
  • querySelectorAll(): 여러 개 선택
const title = document.getElementById("main-title"); const items = document.querySelectorAll(".list-item");

✏️ 4. DOM 조작 (속성, 내용, 스타일)

  • 텍스트 변경: textContent, innerHTML
  • 속성 변경: setAttribute(), href, src
  • 스타일 조작: style 속성 사용
const box = document.querySelector(".box"); box.textContent = "새로운 내용"; box.setAttribute("id", "new-box"); box.style.backgroundColor = "lightyellow";

🛠️ 5. DOM 요소 생성 및 삭제

  • createElement(): 요소 생성
  • appendChild(), prepend(): 자식으로 추가
  • remove(), removeChild(): 삭제
const newItem = document.createElement("li"); newItem.textContent = "추가된 항목"; document.querySelector("ul").appendChild(newItem); newItem.remove(); // 제거

🧠 6. 실습 예시: 버튼 클릭 시 문단 추가

<button id="add-btn">추가</button> <div id="container"></div> <script> document.getElementById("add-btn").addEventListener("click", () => { const p = document.createElement("p"); p.textContent = "새 문단이 추가되었습니다."; document.getElementById("container").appendChild(p); }); </script>
