📝 폼 값 읽기 및 검증 실습

📌 1. 폼(Form)이란?

  • 사용자의 입력을 받기 위한 HTML 요소 집합
  • <input>, <textarea>, <select> 등 포함
  • submit 이벤트로 입력값 전송 가능

🎯 2. 값 읽기 방법 (JavaScript)

  • DOM을 통해 입력 필드에 접근
  • .value 속성으로 값 가져오기
const nameInput = document.getElementById("username"); let name = nameInput.value; console.log("입력된 이름:", name);

🛡️ 3. 유효성 검사 (Validation)

  • 값이 비어있는지 확인
  • 이메일 형식, 비밀번호 길이 등 조건 검사
  • 맞지 않으면 경고 메시지 출력 또는 제출 중단
if (name === "") { alert("이름을 입력해주세요."); return; }

🧪 4. 실습 예시: 이름과 이메일 입력

<form id="myForm"> 이름: <input type="text" id="username"><br> 이메일: <input type="email" id="email"><br> <button type="submit">제출</button> </form> <script> document.getElementById("myForm").addEventListener("submit", function(e) { e.preventDefault(); // 기본 제출 방지 const name = document.getElementById("username").value; const email = document.getElementById("email").value; if (name === "") { alert("이름을 입력하세요."); return; } if (!email.includes("@")) { alert("유효한 이메일을 입력하세요."); return; } alert("폼 제출 완료!\n이름: " + name + "\n이메일: " + email); }); </script>

📌 5. 참고: HTML 자체 검증 속성

  • required: 필수 입력
  • minlength, maxlength: 길이 제한
  • pattern: 정규표현식으로 패턴 제한
<input type="text" required minlength="2" pattern="[a-zA-Z]+">
