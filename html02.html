💻 HTML 주요 태그 요약

🏷️ 1. 제목 태그 (`<h1>` ~ `<h6>`)

  • <h1>: 가장 큰 제목
  • <h6>: 가장 작은 제목
  • 페이지 구조 및 SEO에 중요
<h1>제목 1</h1> <h2>제목 2</h2> ... <h6>제목 6</h6>

📄 2. 문단과 텍스트 태그

  • <p>: 문단 (paragraph)
  • <br>: 줄바꿈 (종료 태그 없음)
  • <strong>, <em>: 강조 (굵게, 기울임)
<p>이것은 문단입니다.</p> <p>줄바꿈은<br>이렇게 합니다.</p> <strong>중요한 텍스트</strong> <em>기울어진 텍스트</em>

🔗 3. 링크와 이미지

  • <a href="URL">: 하이퍼링크
  • <img src="경로" alt="설명">: 이미지 삽입
<a href="https://example.com">사이트로 이동</a> <img src="image.jpg" alt="설명 텍스트">

📋 4. 목록 태그

  • <ul>: 순서 없는 목록
  • <ol>: 순서 있는 목록
  • <li>: 목록 항목
<ul> <li>항목 1</li> <li>항목 2</li> </ul> <ol> <li>첫째</li> <li>둘째</li> </ol>

📦 5. 구획 및 구조 태그

  • <div>: 블록 요소 그룹
  • <span>: 인라인 요소 그룹
  • <header>, <footer>, <main>, <section>: 의미 있는 구역 구분
<div class="box">레이아웃 박스</div> <span style="color:red">강조 텍스트</span> <header>머리말</header> <main>본문 내용</main> <footer>바닥글</footer>

🔘 6. 입력 및 폼 태그

  • <form>: 입력을 받는 폼 영역
  • <input>: 한 줄 입력
  • <textarea>: 여러 줄 입력
  • <button>: 버튼
  • <label>: 입력 필드 설명
<form> <label for="name">이름:</label> <input type="text" id="name" name="name"> <textarea rows="4" cols="30"></textarea> <button type="submit">제출</button> </form>

🧪 7. 기타 유용한 태그

  • <table>, <tr>, <td>, <th>: 표 구성
  • <iframe>: 외부 웹페이지 포함
  • <script>: JavaScript 코드 포함
  • <link>, <style>: CSS 포함
<table> <tr><th>이름</th><th>나이</th></tr> <tr><td>홍길동</td><td>25</td></tr> </table>
