📘 조건문, 반복문, 함수, 이벤트 처리

✅ 1. 조건문 (if / else / switch)

  • if: 조건이 참일 때 실행
  • else: 조건이 거짓일 때 실행
  • else if: 추가 조건
  • switch: 여러 경우의 분기
let score = 85; if (score >= 90) { console.log("A 학점"); } else if (score >= 80) { console.log("B 학점"); } else { console.log("C 학점 이하"); } let fruit = "banana"; switch (fruit) { case "apple": console.log("사과입니다"); break; case "banana": console.log("바나나입니다"); break; default: console.log("알 수 없음"); }

🔁 2. 반복문 (for / while)

  • for: 반복 횟수가 정해졌을 때 사용
  • while: 조건이 참인 동안 반복
  • for...of: 배열 순회
for (let i = 0; i < 5; i++) { console.log("i =", i); } let count = 3; while (count > 0) { console.log("남은 횟수:", count); count--; } let fruits = ["사과", "배", "포도"]; for (let fruit of fruits) { console.log(fruit); }

🔧 3. 함수 (Function)

  • 동작을 정의하고 재사용 가능
  • function 키워드 또는 화살표 함수 사용
  • 매개변수, 반환값 포함 가능
// 기본 함수 선언 function greet(name) { return "안녕하세요, " + name + "님!"; } console.log(greet("홍길동")); // 화살표 함수 const add = (a, b) => a + b; console.log(add(3, 5)); // 8

🖱️ 4. 이벤트 처리

  • 사용자의 행동에 반응하는 코드 작성
  • 예: 클릭, 입력, 마우스 오버 등
  • addEventListener로 이벤트 연결
// 버튼 클릭 시 실행 const button = document.querySelector("button"); button.addEventListener("click", () => { alert("버튼이 클릭되었습니다!"); });
