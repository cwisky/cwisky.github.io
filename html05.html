🧾 HTML Table 태그 및 하위 태그 요약

📌 1. <table> 태그란?

  • 행(Row)과 열(Column) 구조로 데이터를 표현
  • 표 전체를 감싸는 **컨테이너 역할**
<table> ...표 내용... </table>

📋 2. 주요 하위 태그 정리

  • <tr>: 테이블의 한 행 (Table Row)
  • <td>: 일반 셀 (Table Data)
  • <th>: 헤더 셀 (Table Header)
  • <thead>: 표의 머리글 그룹 (Header Section)
  • <tbody>: 표의 본문 영역 (Body Section)
  • <tfoot>: 표의 바닥글 영역 (Footer Section)
  • colspan, rowspan: 셀 병합 속성

📌 3. 기본 구조 예시

<table border="1"> <thead> <tr> <th>이름</th> <th>나이</th> </tr> </thead> <tbody> <tr> <td>홍길동</td> <td>25</td> </tr> <tr> <td>김영희</td> <td>30</td> </tr> </tbody> <tfoot> <tr> <td colspan="2">총 2명</td> </tr> </tfoot> </table>

🔀 4. 셀 병합 속성

  • colspan: 열 병합
  • rowspan: 행 병합
<table border="1"> <tr> <td rowspan="2">이름</td> <td>나이</td> </tr> <tr> <td>25</td> </tr> </table>

✅ 5. 시멘틱 그룹 태그 사용 이유

  • <thead>, <tbody>, <tfoot>를 나누면
  • 스타일 지정, 스크롤 고정 등에 유리
  • 스크린 리더 접근성 향상
