🎨 CSS 핵심 개념 요약

🧷 1. 선택자 (Selector)

  • 태그 선택자: 요소 이름으로 선택 → p { }
  • 클래스 선택자: .클래스명 → .highlight { }
  • 아이디 선택자: #아이디명 → #header { }
  • 자손/자식 선택자: div p / div > p
/* 예시 */ p { color: blue; } .highlight { background-color: yellow; } #header { font-weight: bold; }

🎨 2. 색상 지정 (Color)

  • 이름으로 지정: red, blue
  • Hex 코드: #ff0000
  • RGB: rgb(255, 0, 0)
  • RGBA (투명도 포함): rgba(0, 0, 0, 0.5)
h1 { color: #007bff; background-color: rgba(0, 0, 0, 0.05); }

🔤 3. 글꼴 스타일 (Fonts)

  • font-family: 글꼴 지정 (여러 개 가능)
  • font-size: 글자 크기 (px, em, rem 등)
  • font-weight: 굵기 (normal, bold, 100~900)
  • text-align: 정렬 (left, center, right)
body { font-family: 'Segoe UI', sans-serif; font-size: 16px; font-weight: 400; text-align: left; }

📦 4. 박스 모델(Box Model)

  • 모든 HTML 요소는 사각형 박스 형태로 구성
  • 구성 요소: Content → Padding → Border → Margin
  • 시각적으로 레이아웃을 조절하는 핵심 구조
div { padding: 10px; border: 2px solid #ccc; margin: 20px; }

↔️ 5. 여백: Margin vs Padding

  • Margin: 요소 바깥 여백 (다른 요소와의 거리)
  • Padding: 요소 안쪽 여백 (내용과 테두리 사이)
.box { margin: 20px; /* 외부 간격 */ padding: 15px; /* 내부 간격 */ }
