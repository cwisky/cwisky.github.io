📝 Form 태그 & 🧾 Table 태그 요약

📝 1. Form (폼) 태그란?

  • 사용자로부터 **입력 데이터를 수집**하는 영역
  • <form> 안에 다양한 입력 요소 포함
  • 주요 속성:
    • action: 데이터를 전송할 URL
    • method: 전송 방식 (GET / POST)

📌 주요 입력 태그

  • <input>: 텍스트, 비밀번호, 이메일 등
  • <textarea>: 여러 줄 텍스트
  • <select>: 드롭다운 목록
  • <button>: 버튼
<form action="/submit" method="post"> <label for="name">이름:</label> <input type="text" id="name" name="name"><br> <label for="msg">메시지:</label> <textarea id="msg" name="msg"></textarea><br> <button type="submit">전송</button> </form>

🧾 2. Table (표) 태그란?

  • 데이터를 **행(Row)**과 **열(Column)**로 표현
  • <table>: 표 전체
  • <tr>: 행 (table row)
  • <th>: 제목 셀 (table header)
  • <td>: 일반 셀 (table data)

📌 기본 구조

<table border="1"> <tr> <th>이름</th> <th>나이</th> </tr> <tr> <td>홍길동</td> <td>25</td> </tr> </table>
