📦 Flexbox 속성과 사용법 요약

📌 1. Flexbox란?

  • CSS에서 **레이아웃을 쉽게 구성**하기 위한 방법
  • 가로 정렬, 세로 정렬, 간격 조절 등을 간단하게 구현
  • 부모 요소에 display: flex를 설정하면 적용
.container { display: flex; }

🔀 2. 주요 속성 - 부모 요소 (컨테이너)

  • flex-direction: 방향 설정 (row, column)
  • justify-content: 주축 정렬
  • align-items: 교차축 정렬
  • flex-wrap: 줄 바꿈 허용
  • gap: 아이템 간 간격
.container { display: flex; flex-direction: row; justify-content: space-between; align-items: center; flex-wrap: wrap; gap: 20px; }

🎯 3. 주요 속성 - 자식 요소 (아이템)

  • flex: 성장, 축소, 기준 너비 설정 (단축 속성)
  • align-self: 개별 아이템의 세로 정렬
  • order: 아이템 순서 재배치
.item { flex: 1; /* 같은 비율로 분배 */ align-self: flex-end; order: 2; }

📏 4. flex-direction 종류

  • row: 기본값, 가로 정렬 (왼→오)
  • row-reverse: 가로 반대 (오→왼)
  • column: 세로 정렬 (위→아래)
  • column-reverse: 세로 반대
.container { display: flex; flex-direction: column; }

🧭 5. justify-content (가로 정렬)

  • flex-start: 왼쪽 정렬
  • flex-end: 오른쪽 정렬
  • center: 가운데 정렬
  • space-between: 양쪽 정렬 + 간격 분배
  • space-around: 양쪽 여백 포함

📐 6. align-items (세로 정렬)

  • stretch: 기본값, 높이 늘림
  • flex-start: 위쪽 정렬
  • flex-end: 아래쪽 정렬
  • center: 수직 중앙 정렬
  • baseline: 글꼴 기준선 정렬

💡 7. flex 단축 속성

  • flex: grow shrink basis 형태
  • 예: flex: 1 1 100px
  • 또는 flex: 1 만 사용해 동일 비율 분배
.item { flex: 2 1 200px; }

📚 8. 예시: 가로 정렬 레이아웃

.container { display: flex; justify-content: center; align-items: center; height: 300px; gap: 10px; } .item { background: #d0ebff; padding: 20px; flex: 1; text-align: center; }
