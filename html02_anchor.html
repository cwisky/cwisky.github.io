하이퍼링크의 다양한 형식

1. 외부 웹사이트 링크

외부 사이트로 이동하는 기본적인 하이퍼링크입니다.
예시: 구글로 이동
코드: <a href="https://www.google.com" target="_blank">구글로 이동</a>

2. 같은 페이지 내 앵커 링크

문서 내 특정 위치로 이동할 수 있습니다.
예시: 맨 아래로 이동
코드: <a href="#bottom">맨 아래로 이동</a>

대상 위치에는 id 속성이 있어야 합니다: 

<div id="bottom">여기가 맨 아래입니다</div>

3. 이메일 링크

사용자의 이메일 클라이언트를 열고 메일을 작성하도록 유도합니다.
예시: 메일 보내기
코드: <a href="mailto:example@example.com">메일 보내기</a>

4. 전화 걸기 링크

스마트폰에서 클릭하면 전화 앱이 실행됩니다.
예시: 010-1234-5678 전화 걸기
코드: <a href="tel:01012345678">010-1234-5678 전화 걸기</a>

5. 파일 다운로드 링크

클릭 시 파일이 다운로드됩니다. (브라우저 설정에 따라 다를 수 있음)
예시: PDF 파일 다운로드
코드: <a href="sample.pdf" download>PDF 파일 다운로드</a>

6. 새 창에서 열기 vs 현재 창에서 열기

  • 현재 창: <a href="...">
  • 새 창: <a href="..." target="_blank">

※ 외부 링크는 보통 target="_blank"로 설정하여 새 창에서 여는 것이 일반적입니다.
